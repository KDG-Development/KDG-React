# React Context Utilities

## `useAbortControllerSignal()`
Used to abort network requests, integrates into http utilities
## `useAppNavigation()`
Used to handle consistent navigation
## `usePagination(initialState)`
Used for managing pagination state
## `useNotification()`
Used for managing growler notificaiton state
