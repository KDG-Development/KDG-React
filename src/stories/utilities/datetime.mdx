# DateTime utilities

## `DateUtils.Locale`
Used for formatting values

# `DateUtils.LocalDate`

## `DateUtils.LocalDate.now`
Returns current date
## `DateUtils.LocalDate.parse`
Parse a LocalDate from a given string
## `DateUtils.LocalDate.toDate`
Convert a LocalDate to a Date object
## `DateUtils.LocalDate.fromDate`
Convert a Date object to a LocalDate
## `DateUtils.LocalDate.format`
Formats a LocalDate object to the given format enum or string
