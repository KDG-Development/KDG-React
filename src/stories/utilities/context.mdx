# React Context Utilities

## `contextBuilder(T)`
Used to compose react context from a given type
## `contextDispatchBuilder(T)`
Used to create a dispatcher for updating context value
## `ContextProvider(React.PropsWithChildren<ContextProviderType<T>>)`
Used to compose a provider for wrapping applicable components dependant on context
